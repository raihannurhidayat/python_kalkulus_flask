Fungsi Kubik


Fungsi kubik merupakan fungsi polinomial berderajat 3. Jadi grafik fungsi kubus dapat mempunyai maksimal 3 akar. yaitu, ia dapat memotong sumbu x maksimal pada 3 titik. Karena akar kompleks selalu berpasangan, fungsi kubik selalu mempunyai 1 atau 3 angka nol real. Tidak boleh ada 2 angka nol yang sebenarnya.


Mari kita pelajari lebih lanjut tentang fungsi kubik beserta domainnya, jangkauannya, dan proses pembuatan grafiknya. Mari kita pelajari juga cara mencari titik kritis dan titik belok fungsi kubus dan lihat juga perilaku akhirnya.


Apa itu Fungsi Kubik?


Fungsi kubik adalah fungsi polinomial berderajat 3 dan berbentuk f ( x ) = ax 3 + bx 2 + cx + d , dengan a, b, c, dan d adalah bilangan real dan a ≠ 0. Fungsi kubik dasar fungsi (yang juga dikenal sebagai fungsi kubus induk) adalah f ( x ) = ax 3 . Karena fungsi kubik melibatkan polinomial berderajat ganjil, maka fungsi tersebut mempunyai paling sedikit satu akar real. Misalnya, hanya ada satu bilangan real yang memenuhi ax 3 = 0 (yaitu x = 0) sehingga fungsi kubik f ( x ) = ax 3 hanya mempunyai satu akar real (dua akar lainnya adalah bilangan kompleks). Asimtot Fungsi Kubus


Asimtotnya selalu sesuai dengan nilai yang dikecualikan dari domain dan rentang . Karena domain dan rentang fungsi kubik adalah himpunan semua bilangan real, tidak ada nilai yang dikecualikan dari domain atau rentang tersebut. Oleh karena itu, fungsi kubik tidak memiliki asimtot vertikal maupun asimtot horizontal .


Perpotongan X Fungsi Kubik


Perpotongan x suatu fungsi juga dikenal sebagai akar (atau) nol. Karena derajat fungsi kubik adalah 3, maka fungsi tersebut dapat memiliki maksimal 3 akar. Karena akar kompleks dari suatu fungsi selalu berpasangan, suatu fungsi akan selalu memiliki 0, 2, 4, ... akar kompleks. Jadi suatu fungsi bisa mempunyai 0 atau dua akar kompleks. Jadi, ia mempunyai satu atau tiga akar real atau titik potong x. Untuk mencari titik potong x suatu fungsi kubik, kita cukup mensubstitusikan y = 0 (atau f(x) = 0) dan mencari nilai x.



Bilangan kompleks tidak bisa berupa titik potong x. Oleh karena itu, f(x) hanya memiliki satu titik potong x yaitu (4, 0).


Y-Intercept Fungsi Kubik


Fungsi kubik selalu memiliki tepat satu titik potong y. Untuk menemukan titik potong y dari fungsi kubik, kita cukup mengganti x = 0 dan mencari nilai y.


Contoh: Untuk mencari titik potong y dari f ( x ) = x 3 - 4x 2 + x - 4 , substitusi x = 0. Maka f ( x ) = 0 3 - 4(0) 2 + 0 - 4 . Oleh karena itu, titik potong y dari fungsi ini adalah (0, -4).
